What is the difference between var and let?

var declares a variable similar to let, however, it has no block scope and declarations are processed at the function start meaning that variables are defined from the beginning of the function while let cannot be redeclared and it has block scope, which means that the variable is only accessible within the curly braces in which it is declared.

What is the difference between var and const?

var are function scoped or global scoped that can be hoisted(raised to the top of the function) while const are blocked scoped variables that are constant and cannot be reassigned once it has been assigned.

What is the difference between let and const?

Both let and var are blocked scoped, howere, let can be reassigned but not redeclared while const cannot be reassigned or redeclared.

What is hoisting?

hoisting allows you to use functions and variables before they are declared.
